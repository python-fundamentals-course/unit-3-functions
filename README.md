# Unit 3: Functions & Code Organisation

> **Python Fundamentals**

Writing functions worth keeping, the DRY principle, and building reusable modules.

## New here?

If this is your first time opening a unit repo, start with
[**GETTING_STARTED.md**](GETTING_STARTED.md) — it covers opening this
Codespace, using Jupyter notebooks, doing the exercises, and checking your
work. You don't need to install anything.

## Learning objectives

By the end of this unit you will be able to:

- Identify when a block of code should become a function
- Write functions with parameters, default values, and return values
- Refactor a procedural script into a modular structure
